### Description:

Using ChatGPT API and NodeJs to create an old-style chat application in the theme of Tron. Beginner tier react app. 

## Setup Env
In a new terminal:
npm init -y 
npm install openai dotenv chalk readline-sync

## Run Env
node index.js
